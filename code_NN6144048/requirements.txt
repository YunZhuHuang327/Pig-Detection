#
# 3. transformers 和 timm 用於 DINOv2 backbone
#
# 4. 某些套件可能需要系統級依賴:
#    - opencv-python 可能需要: libglib2.0-0 libsm6 libxext6 libxrender-dev
#    - pycocotools 需要: gcc, python-dev
#
# 5. 如果遇到安裝問題，可以使用 conda:
#    conda install pytorch torchvision torchaudio pytorch-cuda=12.1 -c pytorch -c nvidia
#    conda install -c conda-forge opencv pycocotools
